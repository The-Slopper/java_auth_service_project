# r3-java-auth (Round 3 - advanced)

Checksdor of JWT and management of session in Java. Valida tokens, claims and renew
sessions after login.

## Compilando

```bash
javac TokenVerifier.java && java TokenVerifier
```

Bugs sutis of verification of JWT (confuare of algoritmo, claims, comparison of
signature) and ciclo of vida of session.
